# API SAST - Projeto P2

API desenvolvida para a disciplina, com foco em:
- Integração CI/CD
- Análise SAST com SonarCloud
- Deploy em nuvem
- Uso de Docker e GitHub Actions

## Tecnologias
- Node.js
- Express
- PostgreSQL (nuvem)
- Docker
- GitHub Actions
- SonarCloud
- Render

## GitFlow
O projeto utiliza GitFlow, com as branches:
- main
- develop
- feature/*
